# AstroAI Bots — Netlify Backend

Square subscription backend for the AstroAI Bots landing page.

## Folder Structure
```
netlify/
  functions/
    create-subscription.js
netlify.toml
package.json
```

## Environment Variables
Set these in Netlify dashboard → Site configuration → Environment variables:
- SQUARE_ACCESS_TOKEN
- SQUARE_LOCATION_ID
